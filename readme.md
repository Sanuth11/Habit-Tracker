# 📊 Monthly Habit Tracker

A simple and interactive **Habit Tracking Web App** that helps users track daily habits, monitor progress, and analyze weekly performance.
The app provides visual insights using charts and analytics to encourage consistency and productivity.

---

## 🚀 Features

### ✅ Habit Tracking

* Add new habits easily
* Track habits daily in a **monthly calendar (1–31 days)**
* Only the **current day's habit can be checked**

### 🔥 Streak Tracker

* Automatically calculates **current habit streak**
* Shows how many consecutive days you completed a habit

### 📈 Progress Visualization

* **Daily Progress Bar**
* **Monthly Scatter + Line Chart**
* Displays daily completion percentage

### 📊 Weekly Analytics Dashboard

* Weekly progress percentage
* Visual progress bars for each week

### 🧹 Habit Management

* Remove habits with confirmation popup
* Habit icons automatically assigned (💻 Coding, 💧 Water, etc.)

### 💾 Data Persistence

* Uses **LocalStorage**
* Your habits remain saved after refreshing

### 🔄 Automatic Monthly Reset

* Tracker automatically resets when a **new month starts**

---

## 🖥️ Technologies Used
